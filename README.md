# 简介

Fofa EX 是一款基于 Java Swing 编写的 fofa 图形化客户端，将 fofa api 进行简化操作，搜索结果可实时预览，可编辑、可搜索、可导出；可自定义便捷语法，一键输入自定义语法；支持翻页功能，支持自定义查询条目数量功能，支持iconHash计算功能；



# 启动方式

`java "-Dfile.encoding=UTF-8" -jar .\fofaEX.jar`
